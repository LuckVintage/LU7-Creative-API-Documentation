LU7 Creative API Documentation:

/

Returns a HTML page listing all online players with their name, UUID, health, and location.

Example:

GET https://mcapi.lu7.io

/player/:name

Returns JSON data about a specific player including their name, UUID, health, max health, food level, experience level, total experience, experience progress, game mode, location, active potion effects, ender chest contents, armor inventory, ender pearl cooldown, and player statistics. Limited data is available for offline players.

/version

Returns JSON data containing the current API version.

Example:

GET https://mcapi.lu7.io/version
                {
                    "apiVersion": "1.2"
                }

/onlinePlayers

Returns JSON data listing all online players with their name and UUID.

Example:

GET https://mcapi.lu7.io/onlinePlayers
                [
                {
                  "name": "LuckUnstoppable7",
                  "uuid": "c80cd63c-3afb-43c7-aa52-06b4e515238a"
                },
                {
                  "name": "dante_nl",
                  "uuid": "d3e3b4ac-fbb1-4a84-b7b0-483f0aeba1a8"
                }
                ]

/onlinePlayers/count

Returns JSON data with the current count of online players.

Example:

GET https://mcapi.lu7.io/onlinePlayers/count
                {
                    "count": 2
                }

/player/:name/online

Returns JSON data indicating whether the specified player is online.

Example:

GET https://mcapi.lu7.io/player/LuckUnstoppable7/online
                {
                    "name": "LuckUnstoppable7",
                    "online": true,
                    "uuid": "c80cd63c-3afb-43c7-aa52-06b4e515238a"
                }

/player/:name/plots

Returns JSON data containing plot information for the specified player.

Example:

GET https://mcapi.lu7.io/player/LuckUnstoppable7/plots
                    {
                        "plotCount": 2,
                        "plots": [

                            "id": "0;0"
