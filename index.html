LU7 Creative API Documentation:

Overview:

The current API version is 1.5. It was released on Tue 7 May 2024.

This documentation page was last updated on Tue 7 May 2024 and supports the latest API version.

We are not aware about any bugs or issues with the latest version of the API.

If you make too many requests in a short period of time, the API will respond with {"error":{"code":429,"message":"Rate limit exceeded. Please try again later."}} with a 429 HTTP Status code.

If the requested player has never joined LU7 Creative before, the API will respond with: {"error":{"code":404,"message":"Player not found."}} with a 404 HTTP Status code.

If the requested endpoint is invalid, the API will respond with: {"error":{"code":404,"message":"Endpoint not found."}} with a 404 HTTP Status code.

/version

Returns JSON data containing the current API version.

Example:

GET https://mcapi.lu7.io/version
                {
                    "apiVersion": "1.5"
                }

/onlinePlayers

Returns JSON data listing all online players with their name and UUID.

Example:

GET https://mcapi.lu7.io/onlinePlayers
                [
                {
                  "name": "LuckUnstoppable7",
                  "uuid": "c80cd63c-3afb-43c7-aa52-06b4e515238a"
                },
                {
                  "name": "dante_nl",
                  "uuid": "d3e3b4ac-fbb1-4a84-b7b0-483f0aeba1a8"
                }
                ]

/onlinePlayers/count

Returns JSON data with the current count of online players.

Example:

GET https://mcapi.lu7.io/onlinePlayers/count
                {
                    "count": 2
                }

/player/:name/online

Returns JSON data indicating whether the specified player is online.

If the requested player has never joined LU7 Creative before, the API will respond with: "Player not found." with a 404 HTTP Status code.

Example:

GET https://mcapi.lu7.io/player/LuckUnstoppable7/online
                {
                    "name": "LuckUnstoppable7",
                    "online": true,
                    "uuid": "c80cd63c-3afb-43c7-aa52-06b4e515238a"
                }

/player/:name/activePunishments

Returns JSON data containing information about any active punishments the player has.

If the requested player does not have any active punishments, the API will respond with: "Player has no active punishments." with a 200 Ok HTTP Status code.

If the requested player has never joined LU7 Creative before, the API will respond with: "Player not found." with a 404 HTTP Status code.

A 'dateEnd' value of '0' means the punishment never expires and is permanent.

Example:

GET https://mcapi.lu7.io/player/SubnetSeeker/activePunishments
